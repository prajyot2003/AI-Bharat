# Software Engineer Career Guide

## Overview

Software engineers design, develop, test, and maintain software systems. This is one of the most in-demand roles in the tech industry.

## Key Responsibilities
- Write clean, maintainable code in languages such as Python, JavaScript, Java, or Go
- Collaborate with product teams to understand requirements
- Design system architecture and APIs
- Write unit tests and integration tests
- Participate in code reviews
- Debug production issues and optimize performance

## Required Skills

### Core Technical Skills
- **Programming Languages**: Python, JavaScript/TypeScript, Java, Go, Rust
- **Data Structures & Algorithms**: Arrays, linked lists, trees, graphs, sorting, searching
- **System Design**: Distributed systems, microservices, APIs, databases
- **Databases**: SQL (PostgreSQL, MySQL), NoSQL (MongoDB, DynamoDB)
- **Version Control**: Git, GitHub, branching strategies

### Cloud & DevOps
- Cloud Platforms: AWS, GCP, or Azure
- CI/CD: GitHub Actions, Jenkins, CircleCI
- Containerization: Docker, Kubernetes
- Monitoring: CloudWatch, Datadog, New Relic

## Career Progression

| Level | Years Experience | Typical Salary (US) |
|-------|-----------------|---------------------|
| Junior Engineer | 0–2 years | $80K–$110K |
| Mid-level Engineer | 2–5 years | $110K–$160K |
| Senior Engineer | 5–10 years | $160K–$220K |
| Staff/Principal Engineer | 10+ years | $200K–$400K+ |

## Learning Path

1. **Fundamentals**: CS50 (Harvard), Python/JS crash course
2. **Data Structures**: LeetCode Easy problems, HackerRank
3. **Projects**: Build 3 portfolio projects (CRUD app, API, mini-SaaS)
4. **System Design**: Grokking the System Design Interview
5. **Job Search**: LinkedIn, AngelList, levels.fyi for salary data
